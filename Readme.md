## 🗋 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/3D-Animated-Website.git
   ```
2. Navigate to the project directory:
   ```bash
   cd 3d-animated-website
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the development server:
   ```bash
   npm start
   ```
